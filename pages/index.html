About This Website

This portfolio was created to showcase my work in GIS, coding, and writing. It was built from scratch using HTML, CSS, and JavaScript over a weekend in April 2025.

I used W3Schools to figure out a lot of the JS. When I got stuck, I occasionally turned to other sources linked in the HTML.

The site will continue to evolve as I add new projects. If you open the HTML I have added comments linking to the pages I used for the code.
