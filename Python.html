Sample Code

This function creates a oval cloud shape using multiple overlapping circles. The parameters allow for flexible positioning and scaling:

• t: The Turtle object used to draw
• radius_x: Horizontal radius (width) of the oval
• radius_y: Vertical radius (height) of the oval
• fill_color Fills the oval with the specified color

def draw_oval(t, radius_x, radius_y, fill_color=None):
"""Draw an oval centered at the turtle's current position"""
if fill_color:
t.fillcolor(fill_color)
t.begin_fill()
# Save starting position
# middle of the oval
center_x, center_y = t.xcor(), t.ycor()
# picks up the pen
t.penup()
# makes a full circle
for angle in range(361):
# Convert degrees to radians and compute oval coordinates
x = center_x + radius_x * math.cos(math.radians(angle))
y = center_y + radius_y * math.sin(math.radians(angle))
if angle == 0:
t.goto(x, y)
t.pendown()
else:
t.goto(x, y)

Advanced Text Analysis with SpaCy

In this project I learned how to make a virtual environment in VScode, install python libraries, and apply text analysis techniques. This includes text cleaning and using SpaCy to obtain meaningful word frequency results.

Code Example: Word Tokenization and Normalization

This function processes input text to normalize similar words by converting to lowercase, removing punctuation, numbers, and very short words. If a word passes the tests it is determined to be part of words_normalized and this is returned by the function:

• text: The input string to process
• Converts all text to lowercase to standardize
• Uses a SpaCy nlp model (loaded externally) to tokenize and analyze text
• Filters out newline chars, punctuation, numbers, and words shorter than 3 characters
• Turns remaining words to their base forms
• Returns a list of normalized words

def word_tokenization_normalization(text):
# Convert all text to lowercase to standardize
text = text.lower()
# Process text using SpaCy NLP model
doc = nlp(text)

words_normalized = []
for word in doc:
# Filter out newlines, punctuation, numbers, short words
if word.text != '\n' and not word.is_stop
and not word.is_punct and not word.like_num and len(word.text.strip()) > 2:
# Lemmatize to base form
word_lemmatized = str(word.lemma_)
words_normalized.append(word_lemmatized)

return words_normalized

Exploring a Dataset using Pandas, Matplotlib, Seaborn

In a google Colab we explored how to visualize data. We also got to select and import a real world dataset (in my case NYC baby names).

Sample Code

This script analyzes baby name data from the year 2011, identifies the most popular name, and visualizes the top 50 names using a bar chart:

• Filters the dataset for births in 2011
• Groups by name and sums counts
• Sorts and selects the top name
• Visualizes the top 50 names with seaborn

# Filter the DataFrame for the year 2011
df_2011 = df[df['Year of Birth'] == 2011]

# Group by name and sum the counts
name_total_counts_2011 = df_2011.groupby("Child's First Name")['Count'].sum()

# Sort and get the most popular name
most_popular_name_2011 = name_total_counts_2011.sort_values(ascending=False).index[0]
most_popular_name_count_2011 = name_total_counts_2011.sort_values(ascending=False).iloc[0]

# Print the result
print(f"The most popular baby name in 2011 is '{most_popular_name_2011}' with {most_popular_name_count_2011} occurrences.")

# Visualize the top 50 names
n = 50
top_names_2011 = name_total_counts_2011.sort_values(ascending=False).head(n)

plt.figure(figsize=(12, 6))
sns.barplot(x=top_names_2011.index, y=top_names_2011.values, hue=top_names_2011.index, palette='viridis', legend=False)
plt.title(f'Top {n} Most Popular Baby Names in 2011 (Total Count)')
plt.xlabel('Baby Name')
plt.ylabel()
